Server-Side Polling

Paste your Bitget cookie so the server can fetch data automatically — no need to keep your PC on.

How to get your cookie

Desktop (Chrome / Firefox)

  1. Open bitget.com and log in
  2. Press F12 → Network tab
  3. Refresh the page
  4. Click any request to bitget.com
  5. Find Cookie in Request Headers
  6. Right-click → Copy value
  7. Paste it below

iPhone (easiest: Proxyman)

  1. Install Proxyman (free) from the App Store
  2. Open Proxyman → follow the setup to install its CA certificate
  3. Tap Start to begin intercepting
  4. Open bitget.com in Safari and log in
  5. Back in Proxyman, find any request to bitget.com
  6. Tap the request → Request tab → find Cookie
  7. Long-press → Copy → paste below

iPhone (with a Mac)

  1. iPhone Settings → Safari → Advanced → turn on Web Inspector
  2. Connect iPhone to Mac via USB
  3. On Mac: Safari → Develop → [your iPhone] → [Bitget tab]
  4. Open Network tab → find any request → Cookie header
  5. Copy the value and paste below
