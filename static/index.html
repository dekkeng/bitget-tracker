Server-Side Polling

Paste your Bitget cookie so the server can fetch data automatically — no need to keep your PC on.

How to get your cookie

Desktop (Chrome / Firefox)

  1. Open bitget.com and log in
  2. Press F12 → Network tab → refresh
  3. Click any request to bitget.com
  4. Copy the Cookie request header value

iPhone — Proxyman (easiest)

  1. Install Proxyman from App Store
  2. Start intercepting → open bitget.com in Safari
  3. Find any bitget.com request → Request tab → Cookie header

iPhone — with a Mac

  1. iPhone Settings → Safari → Advanced → Web Inspector on
  2. Connect to Mac via USB
  3. Mac Safari → Develop → [your iPhone] → Network → Cookie header
